# How to commit files
1) Run command 'git diff'
2) Stage all the changed files using 'git add <filename0> .... <filenameN>'
3) Commit the changes to your local repo using 'git commit -m "<Useful commit message. IE: Added in exercise 3.44, finished 3.43>"
4) Push your changes to the remote server using 'git push origin master'
